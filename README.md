# Architect

Architect é uma plataforma desktop para RPG de mesa projetada para centralizar tudo o que jogadores e mestres precisam durante uma campanha.

Diferente de ferramentas tradicionais focadas apenas em fichas ou anotações, o Architect busca oferecer uma experiência imersiva, explorando recursos avançados que aproveitam o poder de aplicações nativas para desktop, incluindo visualização de mapas 3D, gerenciamento de campanhas e automação de mecânicas de jogo.

## Visão do Projeto

A maioria das ferramentas de RPG atuais é construída para navegadores e, consequentemente, limitada pelo ambiente web. O Architect nasce com uma proposta diferente: utilizar tecnologias desktop modernas para entregar funcionalidades mais ambiciosas e performáticas.

O objetivo é criar uma verdadeira central de gerenciamento para campanhas de RPG, permitindo que mestres e jogadores tenham acesso a recursos avançados sem depender de múltiplas aplicações.
